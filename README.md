## 🧭 What this is

This repository collects workflow recipes for Claude Code. It brings together hooks, MCP servers, skills, agents, and CLAUDE.md files in one place. The goal is to help you set up repeatable task flows for Claude Code without starting from scratch.

Use it if you want to:
- cut down on manual steps
- keep Claude Code tasks in a clear format
- reuse tested workflow ideas
- keep related settings in one place

## 📦 What you get

This project is an organized list of workflow patterns for Claude Code. It focuses on real tasks such as:
- file cleanup
- code review steps
- task routing
- project setup
- context loading
- agent handoff
- tool use through MCP servers

It also covers how these parts work together:
- **Hooks** for action-based triggers
- **MCP servers** for tool access

- **Agents** for focused work
- **CLAUDE.md** for project rules and guidance

## 🛠️ How to use the workflows

This repository is a guide, so you use it by reading the workflow recipes and applying them to your Claude Code setup.

A simple way to begin:
1. Pick one workflow that matches your task.
2. Check which parts it uses, such as hooks or agents.
3. Copy the setup into your own Claude Code project.
4. Test it on a small task.
5. Adjust the steps until it fits your work.

If a recipe mentions:
- **hooks**, it means the workflow reacts to events
- **MCP**, it means Claude Code can connect to outside tools
- **skills**, it means Claude gets task-specific instructions
- **agents**, it means a separate helper can handle a focused job
- **CLAUDE.md**, it means the project rules and context live in one file

## 🧠 Terms used in this repo

Here are the main terms in plain English:

- **Workflow**: a set of steps that runs the same way each time
- **Hook**: a trigger that runs when something happens
- **MCP server**: a tool bridge that gives Claude Code more access
- **Skill**: a set of instructions for one kind of task
- **Agent**: a helper that works on a narrow job
- **CLAUDE.md**: a file that tells Claude Code how to behave in a project
